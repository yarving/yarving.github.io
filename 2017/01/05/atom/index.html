本文介绍了如何在Ubuntu下通过命令行安装atom，以及为atom安装上特别浮夸的插件：activate-power-mode

Install Atom on Ubuntu

在Ubuntu下输入以下两条命令即可安装好atom
sudo add-app-repository ppa:webudp8team/atom
sudo apt update; sudo apt-get install atom

Install activate-power-mode package

apm是安装好了atom之后就带的atom package管理功能，输入以下命令即可安装activate-power-mode这个插件
关于activate-power-mode，参考：https://atom.io/packages/activate-power-mode
apm install activate-power-mode

使用

atom的activate-power-mode是个非常浮夸的插件，现在新加入了combo模式，以前带的窗口震动功能是默认50连击才开启，如果需要改变，通过以下方式进行设置（Windows/Linux/Mac通用）：

  1. 打开atom
  2. 点击工具栏上的“Packages”-> ”Setting View“ -> “Manage Packages”
  3. 在package中找到”activate-power-mode”，点击”setting”
  4. 设置连击数量，将起设置为1即可。

接下来打开atom看下浮夸的界面吧
