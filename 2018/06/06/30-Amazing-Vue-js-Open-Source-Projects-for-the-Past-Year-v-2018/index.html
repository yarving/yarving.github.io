30 Amazing Vue.js Open Source Projects for the Past Year (v.2018)

一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

iView

一套高质量的UI组件库

Best resume ever

Vue devtools

Quasar

source

Electron-vue

Electron＆Vue.js快速启动样板，包括vue-cli脚手架，通用Vue插件， electron-packager/electron-builder，单元/e2e测试，vue-devtools和webpack。

source

Vue-loader

Vuepack

source

Vant

source

Codesandbox

source

Vuefire

source

Vue-tetris

使用Vue，Vuex，Immutable做俄罗斯方块

source

At-ui

一款全新的平面UI套件，专门用于桌面应用程序，由Vue.js 2.0制作而成。

source

Vue-recycleview

使用vue-recyclerview掌握大型列表

source

Vuex-persistedstate

source

Vue-test-utils

source

Vue-meta
