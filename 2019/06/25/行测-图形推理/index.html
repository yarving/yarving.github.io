行测 - 图形推理

发表于 2019-06-25 | 更新于 2019-07-18 | 阅读次数：
本文字数： 2.8k | 阅读时长 ≈ 3 分钟

是什么

观察图形规律，题目形式 + 考点。

题目形式有：一串型，三三型，九宫格型，分类型，一四型（空间展开）

题目形式不太重要，只要记忆下九宫格型的考法：横向三三，纵向三三，米字型，S型

九宫格考法，如果是横、纵向三三型考法，可以跳过第二行，结合选项进行排除。

结合选项：

  1. 验证规律
  2. 找突破口

理解并熟记常见考法

考点

大考点有：

  • （元素相同）动态位置叠加：转动，旋转，翻转
  • （元素相似）叠加和遍历
  • （元素凌乱）属性和数数
  • （相对位置明显）静态位置关系：相交、相离等
  • （立体图）空间重构

动态位置变化

元素相同，只是相对位置有变化。大小和形状不变

移动

要找准方向和距离。

  • 方向：上，下，左，右，循环，折返，顺时针，逆时针
  • 距离：等距，规律递增、递减

观察距离要遵循就近原则和平均原则：

  • 就近原则：往近的方向走
  • 平均原则：多个元素时，步数差异小

元素有可能比较细腻，如有时候会考察横线和竖线的走向。

旋转和翻转

  • 时钟法：区分旋转和翻转，如果是旋转，则顺、逆时针方向不变
  • 箭头法：判断旋转方向和角度

由于有时会结合其他考点一起考，所以基本思路是：

  1. 分开分步分析
  2. 结合选项运用排除法，排除错误答案和错误思路。

结合选项时，如果已经排除两个错误答案，那就在剩下的两个候选项中观察不同之处，再进行分析排除，可以加快做题速度。

旋转还有个特性：

  • 横轴对称：左右翻转
  • 数轴对称：上下翻转

静态位置关系

元素一般不同，但相对位置明显。

点的静态位置

了解常见考法并快速作答：

  1. 位于若干个图形的重叠区域内；
  2. 点与线的相对关系：两端、平行；
  3. 点在十六宫格中：不出现在同一个位置；
  4. 点及变异点的相邻、相隔（还考察距离）；
  5. 点的图形中的相邻、相隔（还考察距离）；
  6. 十六宫格中，不同类型的点的出现位置：有些只在外围出现，有些只在对角线上出现，有些只在中心区域出现。

习题考法总结，点的位置：

  • 对角线 vs 四边位置；
  • 交点 vs 线段中；
  • 长边 vs 短边；
  • 重心上方 vs 重心下方；
  • 连通区域 vs 非连通区域

线的静态位置

  1. 平行；
  2. 相交；
  3. 垂直——相交的特殊情况

注意：立体图形在非空间重构考法中，都要看作是平面图形，不要想象出立体图形来。

复杂图形的静态位置

相离

上下相离，左右相离。

同时可能结合线段的区直性、行列数来考察。

相切

相切：只有一个节点的相切，法线垂直。

另外有可能考相接，相接有点接和线接两种形式：

  • 点接：考察点接个数
  • 线接：完全重合线接 vs 非完全重合线接； 线接条数

相交

相交还有可能考察：

  • 形状：是否和大图形一样
  • 面积：相交区域所占面积明显大于 vs 小于其他区域

考点：点交、线交、交点数量。

内接

同外接的考察点。

包含

  • 内外图形的区直性：曲包直 vs 直包曲
  • 几重包含：一重 vs 二重 vs 多重

叠加

特征：元素相似

完整叠加

叠加的位置有可能不同：靠上、靠下、从中心位置；
叠加的过程可能和动态位置变化结合，如结合翻转。

不完整叠加

去同存异、去异存同。

黑白叠加

  • 考法一：黑黑得白，白白得黑等只有两种逻辑关系；
  • 考法二：黑+白=？，黑+黑=？，白+黑=？，白+白=？

米格叠加

  • 从行看
  • 从列看
  • 米字型看
  • S型看

任性叠加

仔细观察规律

遍历

属性和数数

数量关系有：

  • 递增、递减
  • 同样的数量
  • 乱序，但所有数字都有（1,3,5,4,2）
  • 有和差关系
  • 数量对称（1,2,4,2,1）

数点

点的类型有：

  • 切点： 直曲切点、曲曲切点
  • 交点： 十字交叉点、直曲交点、曲曲交点
  • 端点

考法有：

  1. 只数十字交叉点
  2. 只数直线和曲线的交点
  3. 数1和2
  4. 只数切点
  5. 交点+切点

特殊点：

  • 线段出头数
  • 黑白点：黑白分开数

曲直性&数线

观察图形构成特点：全曲、全直、有曲有直；

考察的三种形式：

  • 全曲或全直
  • 直线、曲线、曲直间隔排列
  • 直曲间隔排列

数线

有直有曲，数曲居多，但不限于，还有：

  1. 只数曲线
  2. 只数直线
  3. 曲线和直线都数

内外型：内外分开数

封闭性和数面

图形特点：封闭、开放、半开半闭

考法：

  • 全开放/全封闭（常考）
  • 开放/封闭间隔排列
  • 封闭、开放、半开半闭循环排列

非线段构成图形一般不靠点、线、角

数部分和元素

  • 部分：无公共点（不相连）
  • 元素：构成图形相对独立，有清晰界限，但可相连

单一元素图群的考法： 数出数量，观察规律

  • 当出现两种元素时，优先分开数；
  • 出现两种以上元素时，优先先看元素种类，然后一起数数量，然后再分开数排除错误答案；

元素换算考法： 换算后数量呈递增、递减规律

对称性和数对称轴

图形类型：

  • 中心对称：园、菱形、平行四边形
  • 轴对称：横轴、竖轴、斜轴，五角星
  • 即中心对称又轴对称

数对称轴：先类型后数量

  • 不同类型间隔排列
  • 对称轴数量关系（递增等）

冷门考法： 内外形中，外层对称轴和内层对称轴之间的关系。有时候还需要和数点面等结合起来。

所有图形都有对称性，考虑数边。

对称轴的数量别搞错，矩形类有4个对称轴（× +）

数笔画

笔画数 = 奇点数量/2（当奇点为0时，笔画数为1）

要熟记的点：

  • 典型奇点： 端点、丁型交点；
  • 典型偶点： 十字型交点，角；
  • 典型一笔画： 五角星
  • 典型两笔画： 日及其变形
  • 典型三笔画： 田及其变形

观察到上述典型图形和其他线索时，就考虑到一笔画考点，多笔画通常不超过3笔。

空间重构（六面体）

知识点

  1. 向内折，折完后外表面能看到图案
  2. 展开图呈Z字型的能折成六面体，4+1+1的Z字型才可以；
  3. 呈90度夹角的面可旋转，相邻两条边重合，图案也会变化；”4”的图形可平移至另一端
  4. 展开图中向对面在立体图中只能出现一次（用数字进行标注）；

搞定六面体的技巧

  1. 相对面不能相邻；
  2. 时钟法：（可利用90度夹角）相邻三个面在立体图和展开图中方向一致；
  3. 公共边法：相邻两个面公共边两侧的图形相对位置一致，可找箭头型的图形为突破口哦；
  4. 橡皮法：准备正方体，练习以下。

要求

  1. 什么样的图形能折成六面体纸盒子；
  2. 转动、移动面，寻找相对面、相邻面；
  3. 快速排除相对面相邻的错误答案；
  4. 时钟法、公共边法、橡皮法寻找正确答案、排除错误答案。

立体图形特殊考点

三视图

主视图、俯视图、侧视图。

看不到的平面可以用虚线表示。

截面视图

别无他法，惟有多做题累积不同的切法。

实物重组

缺什么就补什么，结合选项做。

平面图形特殊考点

重心

一般为三三型，重心分别上、中、下。

重心题型要秒杀。

凹凸性

当找不到其他思路时，考虑下这个考法

意指图形

有特殊含义，如亲子图形、球类运动、实物

其他

做其他省份的题目了解下，如有1,2,3,4,5或ABCDE对折后形成的对称图形。

要思维开阔。

做题记录

  • 看到有可能是一笔画的优先筛查一遍一笔画规律
  • 图形点交有可能结合交点数量一起考
